# Does It Matter What You Say? Communication Strategy and Ideology in the Supreme Court

# By Luzciel Ponce-Esquibel and Zachary Serban

This is a final project completed for DATASCI 112: Principles of Data Science class at Stanford University

We analyzed oral argument transcripts from 8,000 Supreme Court cases spanning 1955 to 2023 to examine whether communication style or bench ideology better predicts case outcomes. We found that:

1. Citation rate was the strongest predictor of winning, though barely above random chance (61.3% accuracy)
2. Sentiment and response length show no meaningful difference between winning and losing advocates
3. Ideological features of the bench showed comparable but equally limited predictive power

We conclude that neither communication style nor bench ideology alone is sufficient to predict Supreme Court outcomes.
